Obsdian + Github Action 自动发布博客

配置Github Action

自行创建一个私有仓库(博客源码库)和两个公有仓库(发布库、主题库)

1、 本地执行 ssh-keygen -f hexo-deploy-key -t rsa -C “你的github邮箱” ，会生成私钥 hexo-deploy-key 和 公钥 hexo-deploy-key.pub

2、 在博客源码库中点 Setting -> Secrets -> Action -> New repository secret 将 hexo-deploy-key 复制进去并取名 HEXO_DEPLOY_PRI

3、在发布库中点 ，将 hexo-deploy-key.pub 复制进去并取名 HEXO_DEPLOY_PUB

4、 在博客源码库中新建一个 workflow，取名hexo_deploy.yml；也可以push这个仓库之后新建.github/workflows/hexo_deploy.yml，内容填底下的

Hexo 安装和使用

依次执行安装

brew install node

brew install git

npm install -g hexo-cli

npm install hexo-deployer-git --save

hexo 使用

hexo init blog名称 //回生成hexo目录

修改_config.yml

hexo clean

hexo generate

hexo s //写完可以执行这个 本地查看，没问题就直接git push

hexo deploy //配置了Github Action，这个不需要本地用了
